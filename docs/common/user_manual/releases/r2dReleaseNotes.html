Release Notes¶

Version 4.1.0 (Current)¶

Release date: April 2024

Highlights

  1. Enhanced earthquake event generation toolbox to include simulation of Peak Ground Velocity (PGV) and liquefaction-induced Permanent Ground Deformation (PGD).

  2. Improved BRAILS-Buildings widget for more flexible interaction with BRAILS building inventory generation capabilities, allowing:

    1. Import of footprint and building data from OpenStreetMaps, Microsoft Global Building Footprints dataset, FEMA USA Structures, and user-specified data sources.

    2. Use of National Structures Inventory data or other user-specified data to create a baseline building inventory.

    3. Requesting specific attributes for inventory generation.

  3. Updated the damage and loss engine Pelicun to version 3.3.

Version 4.0.1¶

Release date: January 2024

Highlights

  1. Added support for transportation infrastructure assessment.

  2. Revised earthquake scenario simulation tool interface.

  3. Introduced Brails-Transportation tool for transportation infrastructure inventory generation.

  4. Integration of PreTrained Surrogate Models from EE-UQ for structural analysis.

Version 3.0.0¶

Release date: May 2023

Highlights

  1. Interface revision with a new Tools section and reorganization of existing applications.

  2. Addition of Hazard Occurrence Model for earthquake selection.

  3. Updated input file format to support multiple asset types with multiple asset subtypes.

Version 2.2.0¶

Release date: September 2022

Highlights

  1. Enhanced data acquisition from Census and ARC.

  2. Updated ground motion input options for OpenQuake.

Version 2.1.0¶

Release date: March 2022

Highlights

  1. Regional site response analysis capability for motion translation from rock to surface.

  2. Hazard import as raster.

  3. Custom fragility function support in Pelicun DL framework.

  4. Tsunami hazard support.

  5. Import population demographics from census and integration with building inventory.

Version 2.0.0¶

Release date: December 2021

Highlights

  1. Integration of QGIS into R2D, enabling:

    1. Loading of various GIS files for visualization and information extraction.

    2. Post-process results based on advanced queries and expression filters.

    3. External database joining to GIS features for enhanced pre- and post-processing.

    4. Export of GIS layers in multiple formats for external GIS application viewing.

  2. Advanced query and expression-based asset filtering for analysis inclusion.

  3. OpenQuake source selection tool enhancements:

    1. Scenario-Based and Classical Probabilistic Seismic Hazard Analysis using OpenQuake for intensity measure calculation.

  4. Building inventory import widget supporting ShapeFile format.

Note: Version updates to 2.0.0 are critical as older versions will not run on design-safe due to backend changes.

Version 1.0.0¶

Initial Release for earthquake impact assessment on regions:

  1. Earthquake Scenario application for ground motion selection based on PEER NGA & user-defined inputs.

  2. User-defined building inventory input.

  3. Building modeling with MDOF-LU.

  4. Building loss estimation utilizing Pelicun and Hazus fragilities and consequence functions.
