Release Notes

Version 4.1.0

Release date: Sept. 2024

Highlights

  1. Domain Reduction Method option added in tools to create events for buildings given large physics- based simulations.

  2. ShakerMaker option added to tool to perform earthquake rupture simulations.

  3. OpenSees@DesignSafe option added in tools to allow users to run OpenSees, OpenSeesMP, OpenSeesSP, and OpenSeesPy simulations utilizing TACC HPC resources from their desktop.

Version 4.0.0

Release date: Aug. 2024

Highlights

  1. A needed release for changes required to interact with DesignSafe and new TapisV3 interface.

  2. Due to AI generated spam on message board, users now directed to post questions using github discussions.

Version 3.6.0

Release date: June. 2024

Highlights

  1. Added option to allow different events in multi-model and multi-fidelity simulations

Version 3.5.0

Release date: March. 2024

Highlights

  1. Physics-based ground motion simulations generated by the M9 project.

Version 3.4.0

Release date: October 2023

Highlights

  1. Multi Fidelity Monte Carlo (MFMC) method

Version 3.3.0

Release date: March 2023

Highlights

  1. Multi-model uncertainty propagation options for Modeling (SIM) and Analysis (FEM) tabs

  2. Gaussian process surrogate modeling

Version 3.2.0

Release date: September 2022

Highlights

  1. New Structural Information Model Generators

    1. Concrete Building Model: automatic concrete building design and model generation

    2. MDOF-LU: MDOF shear building model built using user-supplied Hazus data

  2. Moving damping ratio specification from FEM to SIM panels.

  3. Improvements to the message area

  4. Decoupling the output units defined in GI with the units used in the simulation.

Version 3.1.0

Release date: July 2022

Highlights

  1. New Surrogate option in defining spectrum in PEER event

  2. Fixing the radio button in Multiple Event (uncheck is now enabled)

  3. Improvements to the message area, and minor bug fixes

Version 3.0.0

Release date: March 2022

Highlights

  1. New UQ engine: PLoM

  1. Training surrogate model for defined structural model

  2. Training surrogate using user-supplied response data metric

  3. Generating new realizations of structural responses from trained surrogate

  4. Including user-defined ground motion intensity measures in the modeling training

  1. Site-specific seismic disaggregation

  1. Seismic hazard disaggregation at a given return period and user-defined IM

  2. Target conditional mean spectrum (ASCE7) calculation based on disaggregated mean magnitude and distance

  3. Ground motion selection and scaling to the disaggregation-based target spectrum

  1. Refactored results panel synced from new SimCenterCommon

  1. New data visualization panel

  2. Highlighting data points

Version 2.2.7

Release date: January 2022

Highlights

  1. New Options added for PEER NGA Event: Specifying the fault type, pulse-like feature, and duration filter in ground motion selection Adding the option of “Geometric” suite average for the PEER NGA West2 ground motion selection

  2. In User Defined EDP fixed a bug of using user-specified EDP names

Version 2.2.6

Release date: December 22, 2021

Highlights

  1. Specifying the fault type, pulse-like feature, and duration filter in ground motion selection

  2. Adding the option of “Geometric” suite average for the PEER NGA West2 ground motion selection

  3. Bug fixes:

    1. exporting ground motion names in the “Save Data

    2. using user-specified EDP name

Version 2.2.5

  3. changing the look and feel of the message box location
